Quick Start

Linux/macOS 
curl -fsSL https://raw.githubusercontent.com/graz-dev/kalco/master/scripts/install.sh | bash
macOS/Linux 
brew install graz-dev/tap/kalco
Build from source 
git clone https://github.com/graz-dev/kalco.git
cd kalco
go mod tidy
go build -o kalco ./cmd
